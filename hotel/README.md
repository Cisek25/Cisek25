# Hotel Elegance - Szablon IdoBooking

Elegancki szablon dla hotelu 4-gwiazdkowego.

## Pliki

| Plik | Gdzie wkleić w CMS |
|------|-------------------|
| `head.html` | HEAD |
| `sections.html` | Sekcje BODY |
| `styles.css` | Custom CSS |
| `scripts.html` | Koniec sekcji body |

## Paleta kolorów

- **Primary**: #1A365D (granat)
- **Gold**: #C9A227 (złoto)
- **Tło**: #FFFDF7 (kremowy)

## Sekcje

1. **O Nas** - intro z 4 statystykami
2. **Pokoje** - 4 karty (Standard, Superior, Deluxe, Apartament)
3. **Udogodnienia** - 8 ikon (restauracja, SPA, fitness, parking...)
4. **Lokalizacja** - mapa + odległości
5. **Galeria** - 8 zdjęć z lightbox
6. **Opinie** - 3 karty
7. **CTA** - rezerwacja

## Fonty

- **Playfair Display** - nagłówki (elegancki serif)
- **Inter** - tekst (czytelny sans-serif)
