# Pensjonat Pod Jodłami

Tradycyjny polski pensjonat górski - kompletna strona internetowa.

## Kolory

- **Brown Primary**: #8B4513
- **Forest Green**: #2F5233  
- **Cream**: #F5E6D3
- **Sienna**: #A0522D

## Czcionki

- **Nagłówki**: Playfair Display
- **Tekst**: Lato

## Struktura

```
pensjonat-pod-jodlami/
├── css/
│   └── pensjonat-styles.css
├── html/
│   └── complete-pensjonat.html
├── ULTIMATE-PENSJONAT-CSS.css  ← Wklej do Global CSS
└── README.md
```

## Instalacja

1. Wklej `ULTIMATE-PENSJONAT-CSS.css` do Global Custom CSS
2. Użyj HTML z `complete-pensjonat.html` jako template

## Funkcje

- Kompletna strona główna z hero
- Sekcja pokoi z cenami
- Atrakcje turystyczne
- Formularz kontaktowy
- Responsywny design
- Tradycyjny góralski styl
